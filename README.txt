# User Modeling Python Starter Application


## Files

The User Modeling Ruby Sinatra starter application has files as below:

*   app.rb

    This file contains Ruby code for your application written using Sinara API

*   Gemfile

    This includes all dependent Gem of your application

*   templates/

	This directory contains the views of the application. It is required by the mako template.

*   public/

	This directory contains public resources of the application.
