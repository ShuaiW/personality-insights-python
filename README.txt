# Personality Insights Python Starter Application


## Files

The Personality Insights Python starter application has files as below:

*   server.py

    This file contains all the Python code for your application.

*   requirements.txt

    This lists the Python module dependencies for the Buemix buildpack.

*   templates/

	This directory contains the views of the application. It is required by the mako template.

*   public/

	This directory contains public resources of the application.
